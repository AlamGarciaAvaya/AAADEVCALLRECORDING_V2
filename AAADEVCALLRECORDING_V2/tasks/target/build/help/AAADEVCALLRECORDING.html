Call Recording

Definición de la tarea

La tarea Grabar conversación habilita / comienza a grabar la llamada. Tenga en cuenta que la tarea Grabar participante no puede funcionar en la etapa de alerta de llamada, por ejemplo, no se puede colocar inmediatamente después de Permitir llamada, Reenviar llamada, etc.

Esta es una tarea síncrona por defecto. Puede ser una tarea asíncrona si se selecciona la casilla de verificación "Esperar hasta que se complete la grabación".

Propiedades de la tarea

  • Ruta para guardar archivo (obligatoria) El URI del archivo de onda grabado (.wav). Esta es la ubicación donde se grabará la grabación después de que se complete la grabación.

    El esquema de URI debe ser uno de los siguientes http: //, https: //. Al no especificar el nombre del archivo en el URI del archivo la tarea lo generará automáticamente y se agregará al URI del archivo. A continuación se muestra un ejemplo de cada esquema de URI:
    • 2. Http URI: "http://www.mycompany.com/recordings/" ;
    • 3. Https URI: "https://www.mycompany.com/recordings/" Tenga en cuenta que el certificado de confianza de la infraestructura de clave pública (PKI) del servidor HTTP debe estar instalado en el servidor de medios;

    Consulte la documentación de Avaya Aura Media Server para obtener más detalles sobre la administración del almacén de contenido, o para obtener información sobre los formatos de archivo wav compatibles (por ejemplo, mono muestreado de 16 bits a 8 kHz) y certificado de confianza PKI.

  • Duración máxima (Opcional) Especifique la duración máxima de grabación en milisegundos. El valor predeterminado es 3600000ms si no está configurado.

  • Definir tecla para terminar grabación (Optional) Defina la tecla disponible en el teclado que el usuario puede presionar para detener la grabación. Las claves válidas son 0 a 9, * y #.

  • Nombre del archivo (Optional) Seleccione el patrón de nombre de archivo de grabación. Fechay hora_CallingPartyNumber_CalledPartyNumber_CallId ejemplo: 20190111082533_2304_2305_00003002421547220252.wav

  • Espere hasta que se complete la grabación. Seleccione si desea continuar con la siguiente tarea solo después de que se complete la grabación. En algún escenario, el archivo de grabación no necesita ser procesado por este flujo de trabajo en tiempo real, por lo que el Registro La tarea del participante no tiene que esperar hasta que la grabación real esté completamente terminada.

Errores

Los errores se pueden manejar mediante el uso de eventos de límite de error.
